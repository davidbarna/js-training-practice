# JS Training: practice project

**Install project dependencies**

```
$ npm install
```


**Launch the project on dev mode**

 ```
$ npm run start-dev
```

**Open the project in in Chrome**

URL: http://localhost:8080/webpack-dev-server/  

Now you can start coding.
